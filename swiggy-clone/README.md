# вҡҷпёҸ Installation

Clone the repository

```bash
git clone https://github.com/arun-devs/food-app.git
```

Go to the project directory

```bash
cd food-app
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
